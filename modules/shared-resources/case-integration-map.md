### Week 2: Media, Content & Fan Engagement
**CLO 2:** Evaluate Media & Fan Monetization strategies
**Primary Case:** Rogers Communications NHL Deal (BOSM-W2-001 — Custom)
**Supplementary Case:** IPL Media Rights (W36172)

| Module | Case Integration | Purpose | WLO Alignment |
|--------|-----------------|---------|---------------|
| Module 0 (Bridge-In) | Rogers deal preview | Diagnostic question, hook | Pre-assessment |
| Module 3 (Core Content) | IPL Media Rights — Unbundling example | Media valuation framework application | WLO 2.1 |
| Module 4 (Interactive) | Rogers deal data — Streaming Wars context | Platform strategy simulation | WLO 2.2, 2.3 |
| Module 5 (Case Study) | **Rogers NHL Deal (FULL CASE)** | $5.2B deal analysis, 2026 decision | ALL Week 2 WLOs |
| Module 6 (Assessment) | Rogers deal — Mini-case memo | Strategic recommendation | WLO 2.4 |

**Integration Details:**

*Module 0 — Bridge-In*
- Diagnostic: "Rogers paid $5.2B for NHL rights. Was it worth it?"
- Pre-assessment surfaces student assumptions about media valuation
- Creates narrative hook for week's learning

*Module 3 — Core Content Integration*
- Use IPL case (W36172) for unbundling framework
- IPL's digital vs. TV rights separation illustrates modern valuation approaches
- Compare: IPL unbundled ($5.64B), Rogers bundled exclusive ($5.2B)
- Key insight: Different markets, different optimal structures

*Module 4 — Interactive Simulation*
- Streaming Wars Strategy uses Rogers-inspired scenarios
- Platform economics model: subscriber growth vs. content costs
- Students experience tension Rogers faced: invest in streaming vs. protect linear

*Module 5 — Full Case Analysis*
- Rogers case (BOSM-W2-001) provides complete 5-section analysis
- Section 1: Background (Canadian media landscape 2010-2013)
- Section 2: Deal structure and competitive bidding
- Section 3: Strategic rationale and financial projections
- Section 4: 10-year performance analysis (2014-2024)
- Section 5: 2026 decision options
- B-Epilogue: 2025 renewal outcome (for post-discussion)

*Module 6 — Assessment*
- Mini-case memo: Recommend Rogers' 2026 strategy
- Students synthesize all week's frameworks
- Rubric evaluates valuation logic, platform strategy, and strategic synthesis

**IPL Case Deployment:**
- Primary use: Module 3 supplementary reading (optional)
- Key exhibit: Rights unbundling comparison table
- Discussion: Why did Disney/Viacom bid differently for TV vs. digital?
- Connection: Could NHL unbundle Canadian rights similarly?

## Implementation Notes

### Case Distribution Strategy

**Pre-Class Distribution:**
- Full cases should be available 48-72 hours before module
- Teaching notes are instructor-only materials
- Custom cases hosted in course repository

**In-Module Integration:**
- Interactive summaries for students who haven't read full case
- Key exhibits embedded in widgets
- Discussion prompts reference specific case sections

**Assessment Connection:**
- All weekly assessments use case context
- Rubrics reference case analysis skills
- Final project draws on cumulative case learning

### Accessibility Considerations

- All PDF cases should have accessible alternatives (text versions)
- Case exhibits need alt text descriptions
- Audio summaries available for key cases
- Discussion questions provided in advance for preparation
