## SECTION C: Production Guidelines

### Stock Photo Sources (Recommended)
- **Unsplash** (Free, high quality)
- **Pexels** (Free, commercial use)
- **Shutterstock** (Paid, extensive sports library)
- **Getty Images** (Paid, premium quality)
- **iStock** (Paid, good variety)

### Technical Requirements
- **Format:** JPG for photos, PNG for graphics
- **Resolution:** Minimum 1200px wide
- **File Size:** <500KB (optimized for web)
- **Aspect Ratio:** 16:9 preferred for consistency
- **Color Mode:** RGB

### Accessibility Requirements
- All images MUST have descriptive alt text
- Avoid text-only images (use actual text overlays)
- Color contrast minimum 4.5:1 for any text
- Provide long descriptions for complex infographics

### File Naming Convention
```
week[#]-module[#]-[description].jpg
```
Examples:
- `week1-module3-stadium-aerial.jpg`
- `week2-module5-hockey-fans.jpg`
- `week1-infographic-media-timeline.png`
